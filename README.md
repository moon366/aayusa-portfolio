## 🚀 Tech Stack

- **React** + **Vite**
- **TailwindCSS**
- **Framer Motion**
- **Lucide React** & **Iconify** (icons)

## 📁 Project Structure

```
src/
├── assets/          # Images, resume PDF
├── components/      # Reusable UI components
│   ├── ui/          # Small UI primitives
│   └── ...
├── data/            # Personal info, skills, education data
├── pages/           # Section components (About, Skills, Projects, etc.)
└── main.jsx         # Entry point
```

## ✨ Features

- Responsive, animated single-page portfolio
- Spotlight text effect on About section
- Interactive skills grid with hover effects
- Categorized skills (Frontend, Backend, Database, Tools)
- Downloadable resume

## 🛠️ Development

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
